# Coral Island

A tropical island theme inspired by the Coral Island game companion app — warm coral reefs,
deep ocean teal, and sandy cream backgrounds.

## Color Palette

- **Ocean Teal**: `#0f766e` - Primary interactive color (buttons, links, accents)
- **Deep Teal**: `#134e4a` - Dark backgrounds, sidebar, headings
- **Warm Coral**: `#f97316` - CTAs, active states, highlights
- **Sandy Cream**: `#fefce8` - Page backgrounds, cards
- **Teal Light**: `#ccfbf1` - Dividers, borders, hover states
- **White**: `#ffffff` - Card surfaces

## Typography

- **Headers**: Playfair Display Bold (serif — evokes tropical warmth)
- **Body Text**: Inter (400/500/600 — clean and readable)

## Best Used For

Game companion apps, nature/island-themed products, farming sim tools,
tropical brand materials, hobbyist project docs and presentations.
